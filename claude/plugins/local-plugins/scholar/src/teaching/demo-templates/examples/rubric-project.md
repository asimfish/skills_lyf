# Grading Rubric: Final Project

**Course:** STAT-101 - Introduction to Statistical Methods
**Assignment:** Final Statistical Analysis Project
**Total Points:** 100

---

## Project Overview

Students will conduct a complete statistical analysis on a dataset of their choosing, demonstrating mastery of course concepts including descriptive statistics, hypothesis testing, and regression analysis.

---

## Grading Criteria

### 1. Research Question & Introduction (15 points)

| Score | Criteria |
|-------|----------|
| **Excellent (14-15)** | Clear, focused research question. Strong motivation with relevant background. Variables clearly defined. Hypotheses stated appropriately. |
| **Good (11-13)** | Good research question with adequate motivation. Variables defined. Minor issues with hypothesis formulation. |
| **Satisfactory (8-10)** | Research question present but vague. Limited background. Variables partially defined. |
| **Needs Work (5-7)** | Unclear research question. Minimal motivation. Variables not well defined. |
| **Insufficient (0-4)** | No clear research question. Missing background or context. |

---

### 2. Data Description (15 points)

| Score | Criteria |
|-------|----------|
| **Excellent (14-15)** | Complete description of data source, collection method, sample size. All variables described with appropriate types. Limitations acknowledged. |
| **Good (11-13)** | Good data description. Most variables described correctly. Sample size stated. |
| **Satisfactory (8-10)** | Basic data description. Some variables missing details. Source mentioned. |
| **Needs Work (5-7)** | Incomplete data description. Variable types not specified. |
| **Insufficient (0-4)** | Minimal or no data description. |

---

### 3. Descriptive Statistics (20 points)

| Score | Criteria |
|-------|----------|
| **Excellent (18-20)** | Comprehensive summaries (mean, median, SD, range) for all relevant variables. Appropriate visualizations (histograms, boxplots, scatterplots). Clear interpretation of patterns. |
| **Good (14-17)** | Good summaries with appropriate measures. Some visualizations included. Interpretations mostly correct. |
| **Satisfactory (10-13)** | Basic descriptive statistics calculated. Limited visualizations. Surface-level interpretation. |
| **Needs Work (6-9)** | Incomplete descriptive statistics. Few or no visualizations. Minimal interpretation. |
| **Insufficient (0-5)** | Missing descriptive statistics or major calculation errors. |

---

### 4. Statistical Analysis (25 points)

| Score | Criteria |
|-------|----------|
| **Excellent (23-25)** | Appropriate test chosen with clear justification. Assumptions checked and addressed. Correct calculations and interpretation. Effect size reported. All results in context. |
| **Good (18-22)** | Appropriate test selected. Most assumptions checked. Calculations correct. Good interpretation with minor omissions. |
| **Satisfactory (13-17)** | Reasonable test choice but limited justification. Some assumptions checked. Calculations mostly correct. Basic interpretation. |
| **Needs Work (8-12)** | Questionable test choice. Assumptions not checked. Calculation errors present. Interpretation incomplete. |
| **Insufficient (0-7)** | Inappropriate test or major errors. No assumption checking. Incorrect interpretation. |

---

### 5. Results & Conclusions (15 points)

| Score | Criteria |
|-------|----------|
| **Excellent (14-15)** | Clear summary of findings. Results linked to research question. Appropriate conclusions (no over-claims). Limitations discussed. Future directions suggested. |
| **Good (11-13)** | Good summary with clear conclusions. Most limitations mentioned. Results mostly linked to hypothesis. |
| **Satisfactory (8-10)** | Basic conclusions present. Some disconnect from analysis. Limited discussion of limitations. |
| **Needs Work (5-7)** | Weak conclusions not supported by analysis. No limitations discussed. |
| **Insufficient (0-4)** | Missing or incorrect conclusions. Results contradict analysis. |

---

### 6. Presentation & Formatting (10 points)

| Score | Criteria |
|-------|----------|
| **Excellent (9-10)** | Professional formatting. Clear organization with sections. Tables and figures properly labeled. Proper citations. Error-free writing. |
| **Good (7-8)** | Good organization. Most tables/figures labeled. Few grammatical errors. |
| **Satisfactory (5-6)** | Adequate organization. Some formatting issues. Several grammatical errors. |
| **Needs Work (3-4)** | Poor organization. Significant formatting problems. Many errors. |
| **Insufficient (0-2)** | Disorganized. Major formatting issues. Numerous errors affecting readability. |
