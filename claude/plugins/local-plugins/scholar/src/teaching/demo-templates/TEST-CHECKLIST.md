# Teaching Commands Test Checklist

Use this checklist to verify all teaching commands work correctly.

## Prerequisites

- [ ] In demo-course directory
- [ ] `.flow/teach-config.yml` exists
- [ ] Claude Code active

---

## Command Tests

### /teaching:exam

- [ ] `/teaching:exam midterm` generates JSON
- [ ] `/teaching:exam "Chapter 5"` uses custom topic
- [ ] Exam has 10 questions by default
- [ ] Answer key is included
- [ ] Point values sum correctly
- [ ] Mix of question types present
- [ ] LaTeX export produces valid .tex files

### /teaching:quiz

- [ ] `/teaching:quiz "Sampling"` generates questions
- [ ] Mix of question types included
- [ ] Difficulty distribution is reasonable (40% easy, 40% medium, 20% hard)
- [ ] Answer explanations provided
- [ ] Canvas QTI export works

### /teaching:syllabus

- [ ] `/teaching:syllabus` reads from config
- [ ] 15-week schedule generated
- [ ] Grading breakdown matches config
- [ ] Course policies included
- [ ] Learning objectives listed
- [ ] Required materials specified

### /teaching:assignment

- [ ] `/teaching:assignment "Regression"` creates problems
- [ ] Mix of difficulty levels
- [ ] Solutions/answers included
- [ ] Points add up correctly
- [ ] Clear instructions provided

### /teaching:rubric

- [ ] `/teaching:rubric "Project" 100` creates criteria
- [ ] Categories are comprehensive
- [ ] Point allocations sum to total
- [ ] Descriptions are actionable
- [ ] Multiple performance levels defined

### /teaching:slides

- [ ] `/teaching:slides "Probability"` creates outline
- [ ] Sections are well-organized
- [ ] Examples included
- [ ] Practice problems at end
- [ ] Learning objectives stated

### /teaching:feedback

- [ ] `/teaching:feedback sample-student-work.md` works
- [ ] Feedback is constructive
- [ ] Strengths identified
- [ ] Areas for improvement noted
- [ ] Specific suggestions given

---

## Integration Tests

- [ ] Generate exam, then export to LaTeX (3 files: student, key, rubric)
- [ ] Generate quiz, then export to Canvas QTI
- [ ] Config changes reflect in generated output
- [ ] Multiple formats from same source work
- [ ] Math notation renders correctly in all formats

---

## Edge Cases

- [ ] Empty topic: `/teaching:quiz ""` handles gracefully
- [ ] Invalid format: `/teaching:exam --format invalid` shows error
- [ ] Missing config: Command works with defaults
- [ ] Very long topic: `/teaching:slides "Very long topic name..."` works

---

## Performance

- [ ] Exam generation < 30 seconds
- [ ] Quiz generation < 15 seconds
- [ ] Syllabus generation < 20 seconds
- [ ] No memory errors on large exams
