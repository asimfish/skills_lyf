# STAT-101 Demo Course

This is a demo course created by `/teaching:demo` for testing scholar's teaching commands.

## Quick Start

Try these commands in this directory:

### Generate New Materials

```bash
/teaching:exam "Hypothesis Testing"      # Create an exam
/teaching:quiz "Confidence Intervals"    # Create a quiz
/teaching:syllabus                        # Generate syllabus from config
/teaching:assignment "Regression"         # Create homework
/teaching:rubric "Final Project" 100      # Create grading rubric
/teaching:slides "Probability"            # Create lecture slides
/teaching:feedback sample-student-work.md # Generate feedback
```

### Export Formats

After generating content, try exporting to different formats:

- **LaTeX** - For professional printing
- **Canvas QTI** - For LMS import
- **Quarto** - For web publishing
- **JSON** - For programmatic access

## Configuration

Edit `.flow/teach-config.yml` to customize:

- **Course info** - Name, code, instructor
- **Default formats** - Output format preferences
- **Grading weights** - Assignment percentages
- **Topic list** - Course content outline
- **Style** - Tone, notation, examples
